4. Description du Service

INVOOFFICE est une application web fonctionnant entièrement dans le navigateur de l'Utilisateur. Le Service permet de :

  • Saisir des informations d'entreprise, de clients et de lignes de facturation.
  • Générer des documents PDF (factures, devis, bons de livraison, avoirs).
  • Stocker localement les données saisies (entreprise, clients, historique des documents).
  • Exporter une sauvegarde des données au format JSON.

Le Service ne transmet pas le contenu des documents aux serveurs de l'éditeur. La génération des PDF et le stockage des données s'effectuent localement dans le navigateur de l'Utilisateur.
