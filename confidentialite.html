Confidentialité et sécurité de vos données

La confidentialité n'est pas une option pour nous — elle fait partie de la conception même d'INVOOFFICE. Voici comment nous protégeons vos données.

Notre philosophie

Nous pensons que vos documents commerciaux — factures, devis, avoirs — sont des données sensibles. Ils contiennent le nom de vos clients, vos tarifs, le détail de vos prestations. Ces informations vous appartiennent, et nous n'avons aucune raison légitime d'y accéder.

C'est pourquoi INVOOFFICE a été conçu dès le départ pour fonctionner sans jamais envoyer vos données vers un serveur distant. La génération des documents, le stockage des informations, tout se passe dans votre navigateur.

Comment fonctionne INVOOFFICE

Quand vous créez un document avec INVOOFFICE, voici ce qui se passe :

  1. Vous remplissez le formulaire (informations de l'entreprise, client, lignes de facturation).
  2. Quand vous cliquez sur « Générer le document PDF », l'application construit le document au format HTML.
  3. La bibliothèque html2canvas capture ce rendu et le transforme en image.
  4. La bibliothèque jsPDF convertit cette image en fichier PDF.
  5. Le PDF est téléchargé directement sur votre ordinateur.

Tout ce processus s'exécute dans votre navigateur. À aucun moment le contenu de votre document — le nom du client, les lignes de facturation, les montants — n'est transmis à nos serveurs.

Où sont stockées vos données

INVOOFFICE utilise deux mécanismes de stockage intégrés à votre navigateur :

Le LocalStorage

C'est un espace de stockage simple fourni par tous les navigateurs modernes (Chrome, Firefox, Edge, Safari). Il contient :

  • Les informations de votre entreprise (nom, adresse, ICE, IF, etc.)
  • Votre liste de clients enregistrés
  • L'historique de vos documents générés (numéro, date, client, type)
  • Vos préférences (langue, thème clair ou sombre)

L'IndexedDB

Une base de données plus avancée, également intégrée au navigateur. Elle est utilisée comme miroir de sauvegarde du LocalStorage, pour plus de fiabilité.

Ces données sont stockées sur votre appareil, dans votre navigateur. Elles ne sont accessibles que par le site invoooffice.com, et uniquement dans le navigateur que vous utilisez.

Comment supprimer vos données

Vous pouvez à tout moment supprimer l'intégralité de vos données locales :

  • Ouvrez les outils de développement (touche F12).
  • Allez dans l'onglet « Application » (Chrome) ou « Stockage » (Firefox).
  • Dans « Stockage local », trouvez le domaine invoooffice.com.
  • Supprimez toutes les entrées commençant par fb_.
  • Vous pouvez également vider complètement les données de navigation de votre navigateur.

Pourquoi aucun compte n'est nécessaire

La plupart des outils de facturation vous demandent de créer un compte parce qu'ils stockent vos données sur leurs serveurs. Ils ont besoin de vous identifier pour savoir quelles données sont les vôtres.

INVOOFFICE n'a pas de serveur de données. Vos données sont stockées dans votre navigateur. Nous n'avons pas besoin de vous identifier — parce que nous n'avons rien à vous associer. L'absence de compte n'est pas une fonctionnalité optionnelle : c'est une conséquence directe de notre architecture.

  • Simplicité : vous arrivez, vous créez, vous repartez.
  • Rapidité : pas d'écran d'inscription, pas de validation d'email.
  • Confidentialité : pas d'adresse email collectée, pas de mot de passe stocké.

Questions fréquentes

Mes documents quittent-ils mon ordinateur ?

Non. La génération du PDF se fait entièrement dans votre navigateur. Le contenu de vos documents n'est jamais transmis à nos serveurs.

Puis-je supprimer toutes mes données ?

Oui, à tout moment. Vous pouvez supprimer les données directement depuis les outils de développement de votre navigateur (F12 > Stockage local > supprimer les entrées fb_), ou en vidant les données de navigation.

Puis-je utiliser INVOOFFICE sans créer de compte ?

Oui, c'est même le principe. Aucun compte n'est requis. Vous n'avez pas besoin de vous inscrire ni de fournir une adresse email.

Mes documents sont-ils enregistrés sur Internet ?

Non. Vos documents sont stockés uniquement dans votre navigateur. Ils ne sont jamais envoyés à nos serveurs ni sauvegardés sur Internet. Si vous changez d'ordinateur, utilisez la fonction « Exporter une sauvegarde » pour transférer vos données.

Utilisez INVOOFFICE en toute confiance — vos données restent chez vous.
